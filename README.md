# Curso-IA

1. COMO A COMPUTAÇÃO EM NUVEM FUNCIONA

Conceito Central
A computação em nuvem fornece recursos de TI (servidores, armazenamento, redes) sob demanda via internet, eliminando a necessidade de infraestrutura física local.

Características
As a Service:

IaaS (Infraestrutura como Serviço): Ex.: AWS EC2, Azure VMs.

PaaS (Plataforma como Serviço): Ex.: Google App Engine, Heroku.

SaaS (Software como Serviço): Ex.: Gmail, Microsoft 365.

Custos do Modelo Local (On-Premise):

Alto CapEx (investimento em hardware).

Custos contínuos com manutenção, energia e espaço físico.

Custos do Modelo Nuvem:

OpEx (pago conforme o uso, sem custo inicial).

Escalabilidade reduz desperdícios.

Hybrid Model:

Combina nuvem pública (ex.: AWS) e privada (datacenter local).

Ideal para dados sensíveis + flexibilidade da nuvem pública.

Platform:

Ferramentas prontas para desenvolvimento (PaaS) e gerenciamento (IaC).

Regions & Zones:

Regiões: Localizações geográficas (ex.: AWS São Paulo).

Zonas de Disponibilidade: Datacenters redundantes dentro de uma região.


2. COMPUTAÇÃO EM NUVEM

Domínio do Objetivo
Oferecer recursos de TI escaláveis, seguros e econômicos via internet.

Modelos de Nuvem
Pública (AWS, Azure).

Privada (VMware, OpenStack).

Híbrida (combinação dos dois).

CapEx vs. OpEx
CapEx: Investimento inicial em hardware (modelo local).

OpEx: Custos operacionais flexíveis (modelo nuvem).

Escalabilidade
Aumentar/reduzir recursos conforme demanda (ex.: autoscaling).

Elasticidade
Alocação automática de recursos (ex.: AWS Lambda).

Confiabilidade
SLAs garantem tempo de atividade (ex.: 99,9% uptime).

Previsibilidade
Ferramentas de monitoramento (ex.: Azure Cost Management).

Segurança
Criptografia, firewalls, compliance (GDPR, ISO 27001).

Governança
Políticas de acesso, auditoria e conformidade.

Gerenciabilidade
Controle via portal web, CLI ou APIs (ex.: Google Cloud Console).

Destaques: A) A nuvem substitui investimentos pesados em hardware por flexibilidade e custos variáveis. B) Escalabilidade + Elasticidade são essenciais para lidar com demandas dinâmicas. C) Segurança e Governança garantem conformidade e controle. D) Fórmula da Nuvem: Eficiência + Economia + Inovação.
